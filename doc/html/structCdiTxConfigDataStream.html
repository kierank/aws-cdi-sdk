Data Fields

uint16_t stream_identifier
 Used to identify the data stream number associated with this Tx stream. This allows multiple streams to be carried on a single connection and be uniquely transmitted to a unique IP and port.
 
const char * dest_ip_addr_str
 The IP address of the host which is to receive the flow from this transmit stream. NOTE: This must be the dotted form of an IPv4 address. DNS may be supported in the future.
 
int dest_port
 The port number to use at the receiving host. The range of valid values is 1 to 65535, inclusive and must match the value configured for the receiving connection.
 
const char * stream_name_str
 Pointer to name of the stream. It is used as an identifier when generating log messages that are specific to this stream. If NULL, a name is internally generated. Length of name must not exceed MAX_STREAM_NAME_STRING_LENGTH.
