int maximum_payload_bytes
 The maximum number of bytes that can be sent in a packet through this adapter. The number is computed by subtracting the number of bytes required for transmitting a packet through the medium supported by the adapter from the maximum number of bytes in a packet on the medium. In other words, this is the maximum number of bytes that the scatter-gather list provided to CdiAdapterSendPacket() can contain.
